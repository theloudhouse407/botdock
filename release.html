BotDock Release Notes

Version 1.0.0 - Beta Release (August 2025)

Initial Beta launch of BotDock with core features:

  • Generate Discord bot code with customizable commands and events.
  • Folder selection and bot folder management.
  • Dependency checks and install support (discord.js).
  • Start/stop bot process with real-time console output.
  • Error code dropdown with helpful troubleshooting info.
  • Dark theme UI with green glow styling.

Planned for Future Releases

  • More advanced bot configuration options.
  • Support for additional bot libraries and languages.
  • Integrated debugging and logging enhancements.
  • User account management and cloud sync.
  • Custom themes and UI customization.
